🚀 EVITernship Homepage Functional Prompt (React + Animations)

Build a professional EVITernship homepage using React + Framer Motion + Tailwind with smooth animations and working UI interactions.

Ensure no runtime errors occur and all assets are properly imported.

⚠️ Fix Runtime Error

Current error:

[plugin:runtime-error-plugin] project2Img is not defined

Problem:

An image variable is used but not imported.

Fix:

Ensure all project images are imported before usage.

Example:

import projectSocial from "../assets/projects/social-media.png";
import project1Img from "../assets/projects/interview-simulator.png";
import project2Img from "../assets/projects/photofolio.png";

Then use:

const projects = [
  { title: "Social Media App", img: projectSocial, desc: "Dynamic platform with feeds, profiles, and chats." },
  { title: "Interview Simulator", img: project1Img, desc: "AI-powered mock interview practice tool." },
  { title: "PhotoFolio", img: project2Img, desc: "Modern gallery app for professional photographers." }
];

Important:

If an image does not exist, fallback to a placeholder image.

Example:

/assets/projects/placeholder.jpg
🎬 Animations (Must Work)

Use Framer Motion.

Add scroll animations to sections:

fade-up
slide-left
slide-right
scale-in

Example:

<motion.div
initial={{ opacity: 0, y: 40 }}
whileInView={{ opacity: 1, y: 0 }}
transition={{ duration: 0.6 }}
viewport={{ once: true }}
>
🔢 Animated Counter (0 → Number)

Numbers should count up when visible.

Example values:

Projects Built → 200+
Partner Colleges → 100+
Industry Mentors → 50+

Use:
